Media Processing: FFmpeg for format conversion
UI Framework: Tailwind CSS with vanilla JavaScript
Error Handling: Custom error categorization & logging

How to Use DownTube

📥 Basic Download Process

  1. Paste URL: Enter a YouTube video or playlist URL in the input field
  2. Fetch Info: Click "Fetch" to analyze the content and get available formats
  3. Select Format: Choose between MP4 (video) or MP3 (audio only)
  4. Choose Quality: Select your preferred video quality or audio bitrate
  5. Download: Click "Download" to start the process

🎵 Playlist Downloads

• Range Selection: Use the range slider to specify start and end positions
• Concurrent Downloads: Control how many items download simultaneously (1-5)
• Batch Processing: Downloads complete in batches, then start the next batch

⚙️ Settings & Customization

• Output Directory: Set your preferred download folder
• Theme Selection: Choose from multiple color schemes
• Font Options: Customize application fonts
• Concurrent Downloads: Adjust playlist download speed (1-5)

📊 Monitoring & Management

• Downloads Page: Track progress, speed, and ETA for all downloads
• Download History: View completed downloads and access files
• Cancel Downloads: Stop any download at any time

💡 Pro Tips

• Use range selection for large playlists to download specific sections
• Set concurrent downloads to 1 for slower but more stable connections
• MP3 format is perfect for music playlists and audio content
• Check the Downloads page for real-time progress and speed information
• Use the Settings page to customize your download experience
